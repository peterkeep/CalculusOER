Definition 8.1.2. Recursion Relation.

A sequence is defined using a recursion relation is one where the \(n\)th term of the sequence is defined as a function output of the previous term, the \((n-1)\)st term. The sequence also needs some initial term to base the subsequent terms from.
Using notation, we might say that \(a_{n} = f(a_{n-1})\text{.}\)
