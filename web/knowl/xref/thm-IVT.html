Theorem 1.7.3. Intermediate Value Theorem.

If \(f(x)\) is a function that is continuous on \([a,b]\) with \(f(a)\neq f(b)\) and \(L\) is any real number between \(f(a)\) and \(f(b)\) (either \(f(a)\lt L \lt f(b)\) or \(f(b) \lt L \lt f(a)\)), then there exists some \(c\) between \(a\) and \(b\) (\(a\lt c \lt b\)) such that \(f(c)=L\text{.}\)
