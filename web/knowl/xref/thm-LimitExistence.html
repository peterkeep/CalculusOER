Theorem 1.2.4. Mismatched Limits.

For a function \(f(x)\text{,}\) if both \(\displaystyle\lim_{x\to a^-} f(x) \neq \lim_{x\to a^+} f(x)\text{,}\) then we say that \(\displaystyle\lim_{x\to a} f(x)\) does not exist. That is, there is no single real number \(L\) that \(f(x)\) is arbitrarily close to for \(x\)-values that are sufficiently close to, but not equal to, \(x=a\text{.}\)
