Theorem 4.3.3. Second Derivative Test for Local Maximums or Local Minimums.

If \((c,f(c))\) is a critical point of \(f\) with \(f'(c)=0\text{,}\) then we can use the value of the second derivative at \(x=c\) to classify the critical point:
  1. If \(f''(c)\gt 0\text{,}\) then \(f\) is concave up at and around \(x=c\text{,}\) and so the function has a local minimum at \((c,f(c))\text{.}\)
