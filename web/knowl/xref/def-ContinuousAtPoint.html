Definition 1.7.1. Continuous at a Point.

The function \(f(x)\) is continuous at an \(x\)-value in the domain of \(f(x)\) if \(x=a\) if \(\displaystyle \lim_{x\to a}f(x)=f(a)\text{.}\)
