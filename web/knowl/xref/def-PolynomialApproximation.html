Definition 9.1.1. Polynomial Approximation.

If \(f(x)\) is a function that is \(n\)-times differentiable at \(x=a\) (that is, the functions \(f(a)\text{,}\) \(f'(a)\text{,}\) \(f''(a)\text{,}\) ..., \(f^{(n)}(a)\) all exist), then the \(n\)th degree polynomial approximation of \(f(x)\) centered at \(x=a\) is:
\begin{align*} p_n(x) \amp = f(a)+ f'(a)(x-a)+\frac{f''(a)}{2}(x-a)^2+...+ \frac{f^{(n)}(a)}{n!}(x-a)^n\\ \amp = \sum_{k=1}^n \frac{f^{(k)}(a)}{k!}(x-a)^k \end{align*}
